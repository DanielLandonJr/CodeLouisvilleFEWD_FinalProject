  • homeHome Page
    • The home page is designed so that it meets the basic requirements for the project.
    • The project is stored on GitHub and is hosted on GitHub Pages.
    • The bar across the top (with all the numbers) shows how big the screen is. If a box is purple then the screen is at least as large as the number displayed.
    • In the lower left hand corner you will find an indicator that shows the current FPS as well as the number of particles generated.
    • For the most part the page is responsive down to about 300 pixels (menus, formating etc). After that point everything starts to break down.
  • cloudWeather Underground
    • This page demonstrates making a fetch() call and an API to get weather information.
    • The original project is stored on GitHub. Minor changes were made to allow it to function on this site.
    • An API key has been supplied for demonstration purposes. You can change this to a personal API key by modifying the weather_api_KEY.js file.
    • The location is stored in localstorage. If this is the first time running the application it will default to New Albany IN.
    • Selecting the "Change Location" button will allow you to enter a new City/State for display.
  • record_voice_overLanguage Localization
    • Demonstrates how to change the language of the application from a dropdown menu. This would be seen in a desktop application more than a website.
    • The original project is stored on GitHub. Minor changes were made to allow it to function on this site.
    • The various languages are stored in a JSON object and called as needed.

Show Weather UnderGround

This application demonstrates how to make an API call.

Using an API from WeatherUnderground we get the weather information for a specific city and display it for the user. The user is able to change the location and that location will be saved to localStorage. Next time the user starts the app the location that was saved is loaded up and displayed.

This application is actually loaded using a Fetch() call...its an external html file!
